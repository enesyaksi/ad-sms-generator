# Screenshots

This folder contains screenshots for documentation.

## Required Screenshots

Please add the following screenshots:

| Filename | Description | Where to capture |
|----------|-------------|------------------|
| `login.png` | Login page | `/login` page |
| `signup.png` | Signup page | `/signup` page (click "Kayıt Ol") |
| `dashboard.png` | Customer Dashboard with some customers | `/dashboard` after adding customers |
| `dashboard-empty.png` | Empty Dashboard state | `/dashboard` with no customers |
| `sms-generator.png` | SMS Generator form filled | `/` page with form data entered |
| `sms-results.png` | Generated SMS drafts | `/` page after clicking "Oluştur" |
| `settings.png` | User Settings page | `/settings` page |
| `customer-modal.png` | Add/Edit Customer modal | Dashboard → "Yeni Müşteri Ekle" |

## How to Capture Screenshots

1. Open the application in your browser
2. Navigate to the page
3. Use browser screenshot tool or `Win+Shift+S` (Windows) / `Cmd+Shift+4` (Mac)
4. Save with the filename from the table above
5. Place in this folder

## Recommended Settings

- **Resolution**: 1920x1080 or 1280x720
- **Format**: PNG
- **Theme**: Light theme (for documentation readability)
- **Browser**: Chrome or Firefox (hide bookmarks bar)
